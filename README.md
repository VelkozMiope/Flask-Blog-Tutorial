## Iniciando o blog:

Você vai precisar ter o Python instalado na sua máquina
Crie um novo ambiente virtual:
```
py -m venv .venv
```

Para instalar o projeto, utilize:

```
pip install -e .
```

No momento, para iniciar a aplicação use:
```
flask --app flaskr run
```
